# Number Guessing Game

A simple number guessing game built in Python.
The computer picks a random number between 1 and 100 and the player has to guess it.

## How to play
python main.py

Choose a difficulty:
- **Easy** — 10 attempts
- **Hard** — 5 attempts

After each guess the game tells you if the number is too high or too low.

## Built with
- Python 3
- `random` module

## What I learned
- While loops and game logic
- Functions and global variables
- Conditional statements
