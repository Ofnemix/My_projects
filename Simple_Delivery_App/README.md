# Simple Delivery App 

A simple command-line delivery app built in Python.
The user browses the menu, picks multiple dishes and 
receives a full order summary with the total bill at the end.

## How to run
python main.py

## How it works
1. The menu is displayed with prices
2. The user types the name of the dish they want
3. The app asks if they want to add anything else
4. The user can keep adding dishes until they are done
5. At the end the app prints the full order and total bill

## Menu
- Margherita Pizza — $8.50
- Burger — $7.00
- Salad — $5.50
- Fries — $3.00
- Cola — $2.00
- Water — $1.00
